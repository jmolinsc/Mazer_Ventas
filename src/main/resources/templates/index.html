Cambia permisos desde Configuración > Permisos y vuelve a entrar para ver el menú actualizado.

Cómo funciona

  1. Spring Security autentica con `UserDetailsService` usando usuarios guardados en PostgreSQL.
  2. Los roles cargan opciones permitidas desde la relación `rol_opcion`.
  3. El sidebar se construye dinámicamente según el usuario autenticado.
  4. Las rutas también se validan en backend, no solo en la interfaz.
